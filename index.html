冯掌门动力学评分体系 — 基于5个核心指标的综合评分（-100到+100），每个指标反映成分股在对应维度的达标率。
MACD MACD多头

衡量板块内多少成分股的 DIF线在DEA线上方。

  • DIF = 快线（12日EMA - 26日EMA）
  • DEA = 慢线（DIF的9日EMA）
  • DIF > DEA = MACD金叉状态 = 短期动能向上
评分贡献:+10分/只达标率≥50%为✅
MACD MACD红柱

衡量多少成分股的 MACD柱状线 > 0（红色动能柱）。

  • MACD柱 = (DIF - DEA) × 2
  • MACD > 0 = 多头动能区域 = 强势特征
  • 红柱扩大 = 动能增强；红柱缩小 = 动能减弱
评分贡献:+5分/只
均线 5日 > 10日

衡量多少成分股的 5日均线在10日均线上方（短期多头排列）。

  • 5日线上穿10日线 = 短期金叉 = 短线趋势向上
  • 5日线下穿10日线 = 短期死叉 = 短线趋势向下
  • 冯掌门体系：5天线是短期多空分界线
评分贡献:+10分/只
均线 20日 > 60日

衡量多少成分股的 20日均线在60日均线上方（中期多头排列）。

  • 20日线 ≈ 五月线 = 主力进货区间上沿
  • 60日线 ≈ 五周线 = 中期生命线、调整目标位
  • 20日线 > 60日线 = 中期趋势向上，趋势健康
  • 20日死叉60日 = 清仓信号
评分贡献:+10分/只
背离 底背离

冯掌门体系最重要买入信号。判断标准：

  • 股价创新低 + MACD不创新低 = 底背离 ✅
  • 确认流程：底背离 → 阳线穿5天线 → 5金叉10 → MACD变红
  • 板块内出现底背离的个股越多，反转可能性越大
权重:+15分/只（最强信号）
⚠️ 风险信号（扣分项）

• 放量阴线: 量比>1.5且跌幅>2% → 主力出货，-15分
• 5日线死叉10日线: 短期空头 → -10分
• 20日线死叉60日线: 中期空头 → -10分
• MACD绿色: 弱势区域 → -5分
• 远离5天线>8%: 不追高，有回调风险 → -5分
• 缩量下跌: 阴跌风险 → -5分
